Release Notes for @GWTEXT_VERSION@

This release has several new featureand bug fixes. See the GWT issue tracker for the complete list of enhancements and fixes in this release.

New Features

  • Ext 1.1 RC1
    Upgraded to Ext 1.1 RC1
  • ExtElement and Fx
    Completed all API's for ExtElement and Fx (effects)
  • CompositeElement and CompositeElementLite
    Added CompositeElement and CompositeElementList classes
  • MessageBox
    Added MessageBox widget
  • KeyMap and KeyNav
    Added KeyMap and KeyNav classed
  • Layer
    Added Layer class
  • LoadMask
    Added LoadMask class
  • DateUtil
    Added DateUtil class

Breaking API Changes

This release also includes API changes that may require minor tweaks to existing code. Any such changes that affect you should only take a few minutes to rectify.

Grid, EditorGrid, BorderLayout

  • The constructors previously used to accept width and height in the order height, width but they now are in the order width, height.

EditorGridListener

  • The callback methods now have oldValue and newValue to be of type Object. Previously the arguments were of type String.

Renderer

  • Renderer#render is now called with a value of type Object instead of String.

Release Notes for 0.9

This is the first release of GWT-Ext. While a large percentage of the Ext API's are covered, currently there's not much in the form of user documentation, Javadocs and test cases. The idea is to put out a version of GWT-Ext for users to take a look at and provide feedback and report bugs. A large percentage of the Ext API's like data, grid, forms, layoutds, menus have very complete coverage in GWT-Ext. The most notable area that is missing is the Ext Drag and Drop (dd package) API's.

Please join the GWT-Ext Developer Forum to post questions and provide feedback.
